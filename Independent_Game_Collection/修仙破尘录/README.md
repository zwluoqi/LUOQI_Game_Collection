独立游戏，基于利益冲突原则，作品已下架

该作品是一款2D修仙放置类游戏，参考过多款不同类型的放置类、rougolike类游戏，并有服务器支持。这款作品的开发过程中，主要是相关服务器的开发及部署的学习的实践、并伴随有相关数据库的使用。
